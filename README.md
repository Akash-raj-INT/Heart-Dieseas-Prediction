# Heart Disease Prediction

This project builds a Streamlit web app for predicting heart disease risk based on user health inputs.

## Features
- Interactive form for patient details
- Machine learning prediction using a trained model
- Simple and clean Streamlit interface

## Files
- app.py: Streamlit app
- HeartdiseaseFinal.ipynb: notebook used for model development
- KNN_heart.pkl: trained model
- scaler.pkl: feature scaler
- columns.pkl: expected feature columns

## Run locally
```bash
pip install streamlit
streamlit run app.py
```

## Live Demo

## Project Purpose
This app demonstrates how a machine learning model can be deployed as a web application for heart disease risk prediction.
